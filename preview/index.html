Performance hints

Optimization flags for AI-generated code

Lazy loading

Images below fold: loading="lazy". Components: React.lazy + Suspense.

Virtualization

Lists > 50 items: react-window. Tables > 100 rows: virtualize. > 500: paginate.

CSS animations

Prefer transform + opacity. Never animate width/height/margin. Use will-change sparingly.

Rendering

SSR for content/SEO. CSR for dashboards. React.memo on list items and table rows.

Assets

WebP/AVIF images. System font stack. Inline SVG for interactive icons.

Code splitting

Route-level splits. Dynamic import() for modals, heavy charts, settings panels.
